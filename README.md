# Simple Attention on Embedded Environments

This project implements and tests Simple Attention mechanisms on various embedded platforms.

## Building

To build the project, simply run:

```bash
make
```

## Usage

Run the test program with the following command:

```bash
./sima_test <iterations>
```

Where:
- `iterations`: Number of times to run the test

## Example

To run the test 100 times:

```bash
./sima_test 100
```
